# 🛒 Nexora

###### A világ legszélesebb kínálatú webáruháza.

## ❗ Szükséges szoftverek

- [Node.JS](https://nodejs.org/)
- [MariaDB](https://mariadb.org/download/)
- [HeidiSql](https://www.heidisql.com/download.php)

## 🖥️ A projekt indítása

1. `npm install` - Telepíti a szükséges csomagokat.
2. `npm run dev` - Elindítja a projektet nodemon segítségéval.
3. `.env` fájl létrehozása.
4. `prisma generate`.
5. `mariadb/heidisql` telepítése
6. `HeidiSql-be` adatbázist létrehozni.

7. A projekt elérhető a [http://localhost:3000/](http://localhost:3000/).
